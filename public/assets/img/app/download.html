App Preview

Para la vista previa de la aplicación, necesitarás crear una captura de pantalla de tu aplicación o usar una imagen representativa. Te sugiero usar una imagen que muestre las características principales de tu aplicación.

Guardar como: app-preview.png

Alternativa para los badges blancos:

Si los badges blancos no funcionan, puedes:

  1. Usar la versión normal de los badges
  2. Editar la imagen en un editor de imágenes (como Photoshop, GIMP o incluso herramientas online)
  3. Aplicar un filtro de color blanco o ajustar el brillo/contraste para obtener el efecto deseado
  4. Guardar la imagen editada con el nombre correspondiente
